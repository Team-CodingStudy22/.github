## 커밋 컨벤션
- 과목명은 하단 참조.

|Type|Usage|Description|
|:---:|:----:|:-----:|
|study|공부한 주제|공부한 과목명을 **영어로** 작성, 내용도 짧게 작성|
|refactor|공부한 주제|기존 파일에 추가적으로 내용을 덧붙인 경우|
|chore|자잘한 수정 내용|오타 수정, 파일명 수정 등|

### 추가사항
